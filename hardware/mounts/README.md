# Hardware Mounts

This directory contains 3D printable mounting solutions for the bed presence detector.

## Available Mounts

### M5Stack Side Mount (`m5stack_side_mount.stl`)

A bedside mount that attaches to a nightstand or bed frame, angling the sensor toward the bed.

**Print Settings**:
- Layer Height: 0.2mm
- Infill: 20%
- Supports: Not required
- Material: PLA or PETG

## Custom Mounts

Feel free to create your own mounts for different use cases! Consider:
- Wall mounts for overhead detection
- Clip-on mounts for bed frames
- Adjustable angle brackets
